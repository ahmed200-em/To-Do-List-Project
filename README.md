## ✨ Features

- 📋 Organize tasks across **To Do**, **Doing**, and **Done** columns.
- 📊 Monitor productivity with a **live analytics sidebar**.
- 📅 Automatically assign **date stamps** to newly created tasks.
- 🌙 Enjoy a modern **dark-themed UI** with **neon-accented** design elements.
- ⚡ Experience a clean and responsive Kanban workflow.

## 🚀 Getting Started

- **Clone the repository**

```bash
git clone https://github.com/ahmed200-em/To-Do-List-Project.git
```

- **Navigate to the project folder**

```bash
cd taskflow
```

- **Run the application (Option 1)**

```bash
open index.html
```

- **Run the application (Option 2)**

```bash
live-server
```
